Estructuras de Datos con Python

En esta unidad solo veremos las estructuras de datos que ya vienen incluidas en python. No programaremos
una estructura de datos desde cero, más bien aprenderemos a declarar y operar estas estructuras.

  1. Listas
    • Crear una lista vacía
      • estructura=[]
    • Crear una lista no vacía
      • estructura=[1,2,3,5,7,11]
  2. Tuplas
    • Crear una tupla no vacía
      • estructura=(1,2,3,5,7,11)
  3. Diccionarios
    • Crear un diccionario vacío
      • estructura={}
    • Crear un diccionario no vacío
      • estructura={nombre:"Martín",edad:16,ci:12341234}
  4. Conjuntos
    • Crear un conjunto vacío
      • estructura=set{}
    • Crear un conjunto no vacío
      • estructura={a,e,i,o,u}
